Dashboard Principal

O dashboard oferece uma visão geral completa do sistema com estatísticas e gráficos em tempo real.

Estatísticas Principais

  • Total de Reclamações: Número total de reclamações no sistema
  • Pendentes: Reclamações aguardando análise
  • Em Andamento: Reclamações sendo resolvidas
  • Resolvidas: Reclamações finalizadas
  • Usuários Cadastrados: Total de usuários registrados
  • Reclamações Hoje: Novas reclamações do dia atual

Gráficos Disponíveis

  • Gráfico de Pizza: Distribuição por status (Pendente, Em Andamento, Resolvida)
  • Gráfico de Barras: Reclamações por categoria

Atividade Recente

Mostra as 10 reclamações mais recentes com:

  • Título da reclamação
  • Nome do usuário e categoria
  • Tempo decorrido desde a criação
  • Ícone colorido por status
Dica: O dashboard é atualizado automaticamente quando você faz alterações no sistema.
3

Gerenciar Reclamações

A seção de reclamações permite visualizar, editar, filtrar e gerenciar todas as reclamações do sistema.

1

Visualizar Reclamações

Clique em "Reclamações" no menu para ver a tabela completa

2

Filtrar Reclamações

Use os filtros para encontrar reclamações específicas:

  • Por Status: Pendente, Em Andamento, Resolvida
  • Por Categoria: Iluminação, Asfalto, Limpeza, etc.
  • Por Data: Selecione uma data específica
3

Ações Disponíveis

Para cada reclamação, você pode:

  • Editar (ícone lápis): Alterar status, categoria, descrição
  • Visualizar (ícone olho): Ver detalhes completos
  • Excluir (ícone lixeira): Remover reclamação

Editando uma Reclamação

Ao editar uma reclamação, você pode:

  • Alterar o título
  • Mudar o status (Pendente → Em Andamento → Resolvida)
  • Modificar a categoria
  • Editar a descrição
  • Atualizar a localização
  • Adicionar resposta oficial como administrador
Importante: Sempre adicione uma resposta oficial ao alterar o status para "Resolvida".

Gerenciar Usuários

Monitore e gerencie todos os usuários cadastrados no sistema.

Informações dos Usuários

Para cada usuário, você pode ver:

  • ID único do usuário
  • Nome completo
  • Email de contato
  • Bairro e cidade
  • Data de cadastro
  • Número total de reclamações feitas
1

Visualizar Detalhes

Clique no ícone "olho" para ver informações completas do usuário

2

Ver Histórico de Reclamações

Na visualização detalhada, veja todas as reclamações do usuário

3

Excluir Usuário

Se necessário, remova usuários e todas suas reclamações

Atenção: Ao excluir um usuário, todas as suas reclamações também serão removidas!
5

Relatórios Diários

Gere relatórios detalhados para acompanhar o desempenho e atividade do sistema.

1

Acesse a Seção Relatórios

Clique em "Relatórios" no menu administrativo

2

Escolha o Tipo de Relatório

Selecione entre:

  • Relatório Diário: Atividades do dia atual
  • Relatório Semanal: Últimos 7 dias
  • Relatório Mensal: Mês atual
  • Período Personalizado: Escolha datas específicas
3

Gerar e Visualizar

Clique em "Gerar Relatório" para ver os dados

Conteúdo dos Relatórios

  • Estatísticas Gerais: Total, pendentes, em andamento, resolvidas
  • Distribuição por Categoria: Quantas reclamações por tipo
  • Lista Detalhada: Todas as reclamações do período
  • Informações de Geração: Data, hora e período analisado
Dica: Para relatórios diários, a data é preenchida automaticamente com o dia atual.

Exportação de Dados

Exporte relatórios em diferentes formatos para análise externa ou arquivamento.

1

Gere um Relatório

Primeiro, crie um relatório na seção anterior

2

Escolha o Formato

Clique em um dos botões de exportação:

  • Exportar PDF: Documento formatado para impressão
  • Exportar Excel: Planilha para análise de dados
3

Download Automático

O arquivo será baixado automaticamente para seu computador

Exportação PDF

  • Documento profissional formatado
  • Inclui cabeçalho com logo e data
  • Estatísticas organizadas
  • Ideal para apresentações e relatórios oficiais

Exportação Excel

  • Planilha com dados estruturados
  • Múltiplas abas com diferentes informações
  • Dados prontos para análise
  • Ideal para manipulação e cálculos
Dica: Os arquivos são nomeados automaticamente com a data para facilitar a organização.
7

Backup e Restauração

Proteja seus dados fazendo backups regulares e saiba como restaurá-los quando necessário.

1

Acesse Configurações

Clique em "Configurações" no menu administrativo

2

Fazer Backup

Clique em "Fazer Backup" para baixar todos os dados

  • Inclui todas as reclamações
  • Inclui todos os usuários
  • Arquivo JSON com timestamp
3

Restaurar Dados

Para restaurar um backup:

  • Clique em "Restaurar"
  • Selecione o arquivo de backup (.json)
  • Confirme a restauração
Importante: A restauração substitui TODOS os dados atuais. Faça backup antes!

Boas Práticas de Backup

  • Frequência: Faça backup diariamente
  • Armazenamento: Guarde em local seguro (nuvem, HD externo)
  • Teste: Teste a restauração periodicamente
  • Versionamento: Mantenha múltiplas versões

Configurações Avançadas

Gerencie configurações avançadas do sistema e manutenção de dados.

Gerenciamento de Dados

Na seção Configurações, você pode:

  • Fazer Backup: Salvar todos os dados
  • Restaurar Dados: Carregar backup anterior
  • Limpar Dados: Remover todos os dados (cuidado!)
PERIGO: "Limpar Dados" remove TUDO permanentemente. Use apenas para reset completo!
1

Limpeza de Dados

Para limpar todos os dados:

  • Faça backup primeiro!
  • Clique em "Limpar Tudo"
  • Confirme duas vezes
  • Todos os dados serão removidos
2

Monitoramento

Monitore regularmente:

  • Número de usuários ativos
  • Volume de reclamações
  • Taxa de resolução
  • Tempo médio de resposta
9

Melhores Práticas

Dicas e práticas recomendadas para administrar eficientemente o Meu Bairro Alerta.

Resposta Rápida

Responda às reclamações em até 24 horas. Atualize o status regularmente para manter os cidadãos informados.

Análise Regular

Gere relatórios semanais para identificar padrões e áreas que precisam de mais atenção.

Comunicação Clara

Use a resposta do administrador para explicar ações tomadas e próximos passos.

Segurança

Altere as credenciais padrão e faça logout sempre que terminar de usar o sistema.

Backup Regular

Faça backup dos dados diariamente e teste a restauração mensalmente.

Organização

Use filtros para priorizar reclamações urgentes e organizar o trabalho por categoria.

Fluxo de Trabalho Recomendado

  1. Manhã: Verificar dashboard e novas reclamações
  2. Priorizar: Atender reclamações urgentes primeiro
  3. Atualizar: Mudar status conforme progresso
  4. Comunicar: Adicionar respostas oficiais
  5. Relatório: Gerar relatório diário
  6. Backup: Fazer backup dos dados
