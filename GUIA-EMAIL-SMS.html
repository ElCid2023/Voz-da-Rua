Guia de Configuração

Email e SMS - Voz da Rua

Sistema Atual: O sistema de email está implementado e funcionando em modo simulação. Para ativar o envio REAL de emails, siga os passos abaixo (leva apenas 5 minutos).

1 Criar Conta no EmailJS

O EmailJS é um serviço gratuito que permite enviar emails diretamente do navegador.

Acessar EmailJS
  1. Clique em "Sign Up"
  2. Preencha seus dados
  3. Confirme seu email
  4. Faça login no dashboard
Plano Gratuito: 200 emails por mês - suficiente para começar!

2 Configurar Serviço de Email

Conecte sua conta de email (Gmail, Outlook, etc.) ao EmailJS.

  1. No dashboard, clique em "Email Services"
  2. Clique em "Add New Service"
  3. Escolha seu provedor (recomendado: Gmail)
  4. Conecte sua conta seguindo as instruções
  5. Copie o Service ID (ex: service_abc123)
Importante: Guarde o Service ID, você vai precisar dele no Passo 5.

3 Criar Template de Email

Configure o modelo do email que será enviado aos usuários.

  1. Clique em "Email Templates"
  2. Clique em "Create New Template"
  3. Cole o template abaixo:

Template do Email

Assunto:

Confirmação de Reclamação #{{protocolo}} - Voz da Rua

Corpo do Email:

Olá {{to_name}}, Sua reclamação foi registrada com sucesso! DETALHES DA RECLAMAÇÃO: ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ Protocolo: #{{protocolo}} Data: {{data}} Título: {{titulo}} Categoria: {{categoria}} Endereço: {{endereco}} CEP: {{cep}} Descrição: {{descricao}} ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ Acompanhe o status da sua reclamação em: {{site_url}} Atenciosamente, Equipe Voz da Rua by Prof. Borges
  1. Clique em "Save"
  2. Copie o Template ID (ex: template_xyz789)

4 Obter Chave Pública

A chave pública permite que seu site se conecte ao EmailJS.

  1. Clique em "Account" no menu
  2. Vá em "General"
  3. Copie a Public Key (ex: abc123xyz789)

5 Configurar no Código

Agora vamos inserir as chaves no código do site.

  1. Abra o arquivo index.html
  2. Procure por "YOUR_PUBLIC_KEY" (linha ~730)
  3. Substitua pelas suas chaves:
// Substituir na linha ~730 emailjs.init({ publicKey: "SUA_PUBLIC_KEY_AQUI", // Cole sua Public Key }); // Substituir na linha ~820 emailjs.send('SEU_SERVICE_ID', 'SEU_TEMPLATE_ID', templateParams)

Exemplo com valores reais:

emailjs.init({ publicKey: "abc123xyz789", }); emailjs.send('service_abc123', 'template_xyz789', templateParams)

6 Testar o Sistema

Vamos verificar se tudo está funcionando corretamente.

  1. Salve o arquivo index.html
  2. Faça commit e push para o GitHub:
git add . git commit -m "Configurado EmailJS" git push
  1. Aguarde 2 minutos para o GitHub Pages atualizar
  2. Acesse: https://elcid2023.github.io/Voz-da-Rua/
  3. Cadastre uma reclamação de teste
  4. Verifique seu email!
Sucesso! Se recebeu o email, está tudo funcionando perfeitamente!

Alternativa: Envio de SMS

Para enviar SMS em vez de email, use o serviço Twilio.

Acessar Twilio

Passos básicos:

  1. Crie conta no Twilio (crédito gratuito para teste)
  2. Obtenha um número de telefone
  3. Copie Account SID e Auth Token
  4. Use a API REST do Twilio no código
Nota: SMS tem custo por mensagem. Email é gratuito até 200/mês.
